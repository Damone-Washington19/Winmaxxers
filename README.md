# Wikipedia Nanotechnology Knowledge Graph Engine

This engine structures a localized **Closed Universe Network Graph** extracted directly from Wikipedia’s core Nanotechnology clusters (`Nanotechnology`, `Nanomaterials`, `Nanoelectronics`, and `Nanomedicine`). 

## Analytics Architecture
By isolating outbound node linkages exclusively to targets residing within our baseline discovery pool, we process structural matrices free from arbitrary global network distortion. 

### Engineered Graph Features
- **PageRank:** Calculated via iterative power methods ($\alpha = 0.85$) to isolate domain prestige.
- **Betweenness Centrality:** Computed to track sparse connectivity tunnels bridging disparate nano-scientific paradigms.
- **Velocity Deltas:** Evaluates historical topology velocity changes to rank emerging, currently low-prestige concepts.
